## 🔍 Search Tips

### Find by Keyword
```bash
# Linux/Mac/Git Bash
grep -r "keyword" docs/

# Windows PowerShell
Select-String -Path docs\* -Pattern "keyword" -Recurse
```

### Find by Category
- Development → `docs/development/`
- Authentication → `docs/authentication/`
- Systems → `docs/systems/`
- Features → `docs/features/`
- Deployment → `docs/deployment/`
- Troubleshooting → `docs/troubleshooting/`

### Find by Feature
- Subscriptions → `systems/SUBSCRIPTION_*`
- Activity Tracking → `systems/ACTIVITY_*`
- Fine System → `systems/FINE_SYSTEM_*`
- i18n → `features/I18N_*`
- OAuth → `features/OAUTH_*`
